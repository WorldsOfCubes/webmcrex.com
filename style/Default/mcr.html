webMCRex: что это?

webMCRex - это доработанная система управления контентом webMCR, выпущенная командой этого проекта. Все права на ядро системы принадлежат NC22.

Что в ней отличается от обычной webMCR? Много чем :) Дизайн, автодонат, наличие некоторых дополнительных функций...

Сколько она стоит? webMCRex распространяется бесплатно по лицензии GPLv3.

Как заполучить webMCRex? Она публично доступна на битбакките.

webMCRex – система управления контентом с открытым исходным кодом ( GPLv3 ). Изначально разрабатывалась как ЛК с основным набором функций для организации онлайн сервера Minecraft.

Основные компоненты:

  • Новостная лента (рейтинг, комментарии, категории новостей)
  • Размещение файлов
  • Система пользователей и групповая политика
  • Панель управления
  • Личные сообщения
  • Автоматическая покупка групп привелегий и разбан
  • Список пользователей
  • Банлист
  • Поддержка WoCAuth
  • [Minecraft] Загрузка скинов \ плащей ( выбор скина по умолчанию в зависимости от пола )
  • [Minecraft] Технические скрипты авторизации, для взаимодействия с Minecraft сервером
  • [Minecraft] Мониторинг серверов (Неограниченное кол-во, протоколы RCON, Query, Simple Query, JSON API)
  • [Minecraft] Совместимость с плагинами авторизации xAuth и AuthMe
